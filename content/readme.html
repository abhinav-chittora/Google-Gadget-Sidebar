What is Google Gadget Sidebar?

Google Gadget sidebar is an extension for firefox that allows a user to put google gadgets right into a sidebar. You can put any google gadget into this sidebar. This way you need not to open a website to access these services

How to add a new gadget to sidebar?

To add a new google gadget to side bar all you need to do is to paste the gadget script into the text box given and click on Add new button, then you will be promted to give your gadget a name. Click on save button to finish adding the gadget. Now to load the gadget you just saved click on the browse button and select the gadget from the list. Then click on ok. Your gadget will be loaded into the sidebar.

How to load a saved gadget into sidebar?

To load a saved gadget into sidebar, click on the browse button, select the gadget of your choice, the gadgets are saved as a HTML file, and click ok. Your gadget will be loaded into sidebar.

How can I set a default gadget for my sidebar?

To set a default gadget for your sidebar,go to Tools Menu, select addons menu item, look for googlegadget 1.0 addon, click on preferences button, a browse window will be oopened, select the gadget you want to set as default and then click ok. The selected gadget will be loaded each time you open browser unless you don't change your preferences again.

I have added a gadget to addon but its not showing in sidebar, why?

The most probable reason for it can be that you might have not pasted the code properly, try pasting the code again. Another reason may be that the gadget would have been removed from the existing url.
